## 📌 Project Overview
This project focuses on designing a **Battery Management System (BMS)** for Electric Vehicles (EV) to monitor battery conditions and prevent overheating. It ensures safety by detecting abnormal temperature levels and activating protective mechanisms.

## ⚙️ Features
- 🌡️ Real-time Temperature Monitoring (DHT11 Sensor)
- 🔌 Voltage & Current Monitoring
- 🌀 Automatic Cooling Fan Activation
- 🚨 Buzzer Alert for Overheating
- 📟 16x2 LCD Display for Live Data
- 📊 Simulated Real-Time Dataset for Analysis

## 📊 Data Analytics Perspective
- Time-series data collected at second-level intervals  
- Threshold-based anomaly detection  
- Temperature vs Current correlation analysis  
- Dataset used for visualization in Power BI / Excel
